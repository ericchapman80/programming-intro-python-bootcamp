# Repository Strategy

This repository is the curriculum and learning repo:

```text
programming-intro-python-bootcamp
```

It contains:

- Phase guides
- Setup instructions
- Cheat sheets
- AI guidelines
- GitHub templates
- Reviewer guidance
- Reflection templates
- Starter app files
- Sample data files

The Track Career Analyzer app lives under `app/` during the bootcamp.

A later separate repository may be created for a polished final app:

```text
track-career-analyzer
```

That split should be treated as a learning exercise around Phase 7 or Phase 8, not as an early requirement.

## Public Repo Rule

This public repository should not contain full answer keys. Instructor notes may contain guidance, checklists, coaching prompts, and review expectations.
