# Phase 00: Developer Workstation

## Goal

Build comfort with the local development tools used in this bootcamp.

## Learn

- What software is
- What programming is
- What Python does
- What VS Code does
- What Terminal does
- What Homebrew does
- What Git does
- What GitHub does
- Local files vs remote repositories

## Do

Install or verify:

- Homebrew
- Python 3
- Git
- VS Code
- VS Code command line tool

Run:

```bash
brew --version
python3 --version
git --version
code --version
pwd
ls
```

## Git Workflow

```bash
git switch -c phase-00-setup
git status
git add .
git commit -m "Complete phase 00 workstation setup"
git push -u origin phase-00-setup
```

## Demo

- Show the version commands.
- Navigate to the project folder.
- Open the folder in VS Code.
- Explain VS Code, Terminal, Python, Git, and GitHub.
- Make one small live change to a Markdown file.

## Good Enough To Move On

The student can open Terminal, navigate folders, open VS Code, run version checks, and explain local vs remote in plain language.
