# Phase 02: Variables, Input, and Athlete Profile

## Goal

Use variables and input to create an athlete profile.

## Learn

- Variables
- Strings
- Numbers
- `input()`
- Type conversion
- Naming variables clearly

## Build

The program should ask for:

- Athlete name
- Graduation year
- Primary event
- Current PR

Then it should display a formatted athlete profile.

## Demo

- Run the program with different input.
- Explain what a variable is.
- Explain why `input()` returns text.
- Make one small live change, such as adding hometown or team name.
