# Phase 03: Conditionals and Goals

## Goal

Teach decision-making in code.

## Learn

- `if`
- `elif`
- `else`
- Comparison operators
- Boolean logic
- Beginner goal tracking

## Build

The program compares a current mark to a goal mark and displays whether the athlete is at, above, or still chasing the goal.

Use beginner-friendly track examples, such as a school PR goal or conference standard.

## Demo

- Test a mark below the goal.
- Test a mark equal to the goal.
- Test a mark above the goal.
- Explain the `if` / `elif` / `else` decision.
