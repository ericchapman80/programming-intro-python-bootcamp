# Phase 04: Lists, Loops, and Multiple Results

## Goal

Work with more than one result.

## Learn

- Lists
- `for` loops
- Accumulating values
- `min()` and `max()` basics
- Repeated input

## Build

The program should:

- Store multiple meet results.
- Display all results.
- Show the best result.
- Show the number of results entered.

## Demo

- Add several results.
- Explain what a list does.
- Explain why loops are useful.
- Make one small live change, such as adding one more result field.
