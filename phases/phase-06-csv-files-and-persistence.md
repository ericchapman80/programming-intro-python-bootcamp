# Phase 06: CSV Files and Persistence

## Goal

Save data so it does not disappear when the program ends.

## Learn

- Files
- CSV format
- Reading files
- Writing files
- Data persistence
- Basic file paths

## Build

The program should:

- Store track results in `app/data/results.csv`.
- Load results from CSV.
- Add a result and save it.
- Display saved results.

## Demo

- Open the CSV file.
- Run the app.
- Add a result.
- Show the CSV changed.
- Explain why saving matters.
