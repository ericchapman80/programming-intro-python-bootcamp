# Phase 08: Streamlit Phone-Friendly App

## Goal

Turn the Python project into a simple web app that can be viewed in a browser and eventually on a phone.

## Learn

- What a web app is
- What Streamlit is
- Inputs
- Buttons
- Tables
- Charts
- Localhost
- Browser-based apps

## Build

Create `app/streamlit_app.py`.

The app should display:

- Athlete profile
- Results table
- Best marks
- Goal progress
- Simple chart

## Run

```bash
cd app
streamlit run streamlit_app.py
```

## Demo

- Run locally.
- Open in browser.
- Show phone-friendly layout.
- Explain the difference between a terminal app and a web app.
